# Сканер анализ защищенности
CLI-утилита, которая использует ZAP для проверки уязвимостей. 
## Перед началом работы:
```sh
go build -o scan main.go
```
## Функционал:
```sh
./scan add -name [name] -url [url]
./scan alerts -name [name]
./scan scan -name [name]
```
